PROJECT ATLAS

You will build an enterprise-grade unified platform combining:

- Social network
- Real-time chat system
- Collaboration suite
- Cloud storage system
- AI assistant system
- Task/project management system
- Analytics platform
- Admin dashboard
- Developer APIs

All features must share:
authentication, database, notifications, search, analytics, logging, and settings.

The system must behave like a unified operating system in the browser.

TECH STACK:

Frontend:
- HTML5
- CSS3
- Vanilla JavaScript (ES2023+)
- Web Components
- Service Workers
- IndexedDB
- Canvas API
- SVG
- PWA support

Backend:
- Node.js
- Express.js
- Socket.IO
- JWT authentication
- bcrypt
- multer
- sharp
- helmet
- cors
- rate limiting

Database:
- MongoDB Atlas
- Mongoose

Storage:
- Cloudinary

Cache:
- Upstash Redis

Deployment:
- Frontend: GitHub Pages
- Backend: Render

AI:
- Gemini API / Groq / Ollama fallback

You must design everything to be modular, scalable, secure, and production-ready.

FILE: 01_ARCHITECTURE.txt
================================================================================

ARCHITECTURE MODEL:

You will use layered architecture:

Client → API → Controllers → Services → Repositories → Database

RULES:

Controllers:
- Handle requests only
- Validate input
- Call services
- Return response

Services:
- Contain business logic only

Repositories:
- Handle database operations only

Models:
- Define schema structure

Middleware:
- Authentication
- Authorization
- Logging
- Rate limiting
- Validation

UTILITIES:
- Reusable helper functions only

REALTIME:
- Socket.IO server must manage:
  - chat
  - notifications
  - presence
  - live updates

BACKGROUND SYSTEM:
- Cron jobs
- Queues for emails, uploads, AI processing
